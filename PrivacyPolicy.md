# Privacy Policy

**Last Updated:** 2026-04-13

## Overview
Focus Pomodoro Blocker is a Chrome extension designed to help users stay focused by combining a Pomodoro timer with temporary website blocking.

We respect your privacy and are committed to protecting your data.

This extension does not collect, store, or transmit any personal data to external servers.

---

## Data Collection

Focus Pomodoro Blocker does **not collect any personally identifiable information (PII)**.

The extension does not track:
- Browsing history
- User identity
- Personal files
- Keystrokes or input data

---

## Data Usage

All data used by the extension is processed **locally within your browser**.

The extension only uses data necessary to provide its functionality, including:
- Timer state (focus/break sessions)
- User-defined blocked website list
- User preferences (e.g., language, durations)

This data is never sent outside the user's device.

---

## Data Storage

The extension uses Chrome's local storage (`chrome.storage.local`) to store:

- Timer settings (focus duration, break duration)
- Blocked website list
- Current timer state
- Language preference
- Local debug logs (for troubleshooting)

All stored data remains **on your device only**.

---

## Data Sharing

Focus Pomodoro Blocker does **not**:
- Sell data
- Share data
- Transfer data to third parties

No user data is transmitted externally.

---

## External Communication

The extension does not communicate with:
- External APIs
- Remote servers
- Analytics platforms

All functionality operates entirely within the browser.

---

## Permissions Usage

The extension uses the following permissions:

### storage
Used to store user settings and timer data locally.

### alarms
Used to manage Pomodoro timer sessions and transitions between focus and break periods.

### declarativeNetRequest
Used to block user-selected websites during focus sessions.

### optional host permissions
Used only for domains explicitly selected by the user.
Permissions are requested at runtime and only applied to those specific sites.

---

## User Control

Users have full control over:
- Which websites are blocked
- When blocking is active
- Timer configuration

Users can disable or uninstall the extension at any time.

---

## Security

All data is stored locally within the browser environment.
No sensitive information is transmitted or exposed externally.
